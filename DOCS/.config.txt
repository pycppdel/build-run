

    -------------------------------------
    |             .config               |
    -------------------------------------

    In this Section, all the configuration details need to be specified.

    Configuration Possible:

    .background                           "PATH"  || ["p1", "p2", "p3", usw.] || "COLOR"
    .autoscroll                           on/off || true/false
    .permanentscroll                      on/off || true/false

    .fullscreen                           on/off || true/false
    .screenwidth                          INT
    .screenheight                         INT

    .engine                               --[As found in Engines.txt]--

    .fps                                  declares the preferred fps

    .icon                                 the icon of your game as .png or .jpg
    .titel                                the titel of your game

    if another <.CONFIG> was to be included, you still need a .config section.

    Inside this section there needs to be written:

    "extern config"

    as the only parameter
